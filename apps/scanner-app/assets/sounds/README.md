# Sound Assets Placeholder

Add the following audio files for scan feedback:

- `success.mp3` - Played on valid ticket scan (short positive chime)
- `warning.mp3` - Played on already-scanned ticket (warning beep)
- `error.mp3` - Played on invalid ticket (error buzz)

Recommended specs:
- Format: MP3 or WAV
- Duration: 0.3-0.5 seconds
- Sample rate: 44100 Hz
